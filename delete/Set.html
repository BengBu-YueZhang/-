Set的遍历操作

Set 结构没有键名，只有键值（或者说键名和键值是同一个值），所以keys方法和values方法的行为完全一致。Set的Key等于Value

  • keys(), 返回key的遍历器, 可以使用for…of方法遍历
  • values(), 返回value的遍历器
  • entries(), 返回键值的遍历器
  • forEach(), Set的迭代器，行为与数组的forEach方法类似

Set的顺序，是插入Set的顺序

Set的iterator接口

Set的iterator接口返回的遍历器，就是values方法返回的遍历器，Set可以直接使用for…of便利

Set的应用

  1. 数组的去重

由于Set没有filter和map方法，可以将Set转换为数组后使用filter和Set方法

  1. 交集, 并集, 差集

WeakSet

WeakSet的值只能是对象，不能是其他的值。WeakSet对象是弱引用

⚠️什么是弱引用, 垃圾回收机制不乎考虑WeakSet对该对象的引用，如果对象被垃圾回收机制所清楚，WeakSet的内容也会被消失。因为WeakSet成员的不确定性，所以WeakSet是不能被遍历的

  2. delete
  3. has

WeakSet无法获取size和forEach属性，所以WeakSet是无法被遍历的
