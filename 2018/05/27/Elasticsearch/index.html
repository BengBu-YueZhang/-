发布于 2018-05-27

前言

Elasticsearch 是一个可伸缩，开源的全文搜索引擎。你可以使用它接近实时地储存，搜索，快速分析大规模的数据。它通常被用作底层的引擎或技术，加强一些具有复杂的搜索特性和需求的应用程序。

Elasticsearch本身非常强大，我自己本身也在学习中，文档中不可避免的会产生错误，希望大家能够指正出来。

基本概念

Index

Index是一些具有相似特征的文档的集合，可以类比为数据中的表的概念。

Type

在未来的Elasticsearch大版本中，会抛弃Type的概念，目前全部设置

Document

Index中的记录被称为Document，Index是由多条Document组成的。

批处理操作

Elasticsearch提供了_bluk可以批量的执行增删改的操作。_bluk提供了以下四种操作

  • index,创建索引,文档ID相同的会产生覆盖
  • update,更新文档
  • delete,删除文档
  • create,创建文档
