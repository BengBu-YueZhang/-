初窥Xterm.js

Posted on 2019-03-10

前言

入职的新公司所在的事业部专注于K12的编程教育。公司项目里有使用xterm.js这个库, 并基于master分支做出了一定的修改。为了尽快的熟悉业务以及公司的代码, 所以这里打算学习xterm.js的文档(粗略的翻译, 方便自己查阅, 凡是保留原文的地方, 是我目前还没有明白具体使用场景和用法的地方)

xterm.js是什么?

xterm是一个使用TypeScript编写的前端终端组件。并在Vscode等热门项目中得到了应用

模块

xterm

包含了xterm.js的类型声明文件d.ts

FontWeight

设置终端的字体粗细

RendererType

设置终端的渲染方式, dom渲染或者是canvas渲染

类

Terminal
构造函数 constructor

创建一个新的Terminal对象

属性 cols

终端窗口的列数, 可以在创建Terminal指定cols

属性 element
属性 markers

Get all markers registered against the buffer. If the alt buffer is active this will always return []

属性 rows

终端窗口的行数, 可以在创建Terminal指定rows

属性 textarea

返回, 接受终端输入的textarea的dom节点

静态属性 strings

Natural language strings that can be localized. (可以本地化的静态语言字符串?)

方法 addCsiHandler

Adds a handler for CSI escape sequences

方法 addDisposableListener

向终端添加事件监听器, 并返回可用于删除事件监听器的对象, 对象中dispose属性的方法可以取消监听。官方文档上并没有说明, addDisposableListener可以添加那一些事件, 我实验了下可以支持focus, blur, keydown等

具体lssues

方法 addMarker

添加标记, addMarker接受一个数字作为参数, 数字表示当前光标到标记y的偏移量

方法 addOscHandler
方法 attachCustomKeyEventHandler
方法 deregisterCharacterJoiner
方法 deregisterLinkMatcher
方法 blur

使终端失焦

方法 clear

清除整个终端, 只保留当前行

方法 selectAll

选择终端内的所有文本

方法 selectLines

选中指定的两个之间的终端文本

方法 clearSelection

清除当前选择的终端(只是清除选择, 而非清除终端)

方法 destroy

销毁终端, 不推荐使用。推荐使用dispose()

方法 dispose

销毁终端

方法 emit
